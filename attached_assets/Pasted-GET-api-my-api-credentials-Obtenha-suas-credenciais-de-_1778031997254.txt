 Guarde Suas Chaves com Segurança:
A private_key é como sua senha bancária. Nunca compartilhe, nunca exponha em código público e sempre armazene em variáveis de ambiente seguras.
Para autenticar requisições à API, você precisa gerar uma assinatura HMAC-SHA256 usando sua private_key. A assinatura é calculada com base no método HTTP, caminho, corpo da requisição e timestamp.

Importante: O timestamp deve estar em milissegundos e não pode ter mais de 5 minutos de diferença do tempo atual do servidor. Isso evita que requisições antigas sejam reutilizadas por atacantes.
A assinatura é uma "prova digital" que garante que a requisição veio realmente de você. Ela é calculada usando a seguinte fórmula:

mensagem = MÉTODO + CAMINHO + CORPO + TIMESTAMP
assinatura = HMAC-SHA256(chave_privada, mensagem)
Explicação simples: Você pega o método HTTP (POST, GET, etc.), o caminho da URL, o corpo da requisição (JSON ordenado) e o timestamp atual. Concatena tudo e usa sua chave privada para criar uma "assinatura digital" única usando o algoritmo HMAC-SHA256.

Exemplo Prático Passo a Passo:
Vamos criar uma assinatura para uma requisição POST de pagamento:

METHOD: "POST"
PATH: "/api/gateway/payments" (sem query parameters, sem barra no final)
BODY: JSON com chaves ordenadas alfabeticamente. Exemplo:
{"amount":100.50,"description":"Pagamento teste"}
 Importante: As chaves devem estar em ordem alfabética! "amount" vem antes de "description"
TIMESTAMP: Timestamp atual em milissegundos (sem aspas)
1705312800000
MENSAGEM FINAL: Concatene tudo na ordem: METHOD + PATH + BODY + TIMESTAMP
POST/api/gateway/payments{"amount":100.50,"description":"Pagamento teste"}1705312800000
 Dica: Use a função HMAC-SHA256 com sua private_key para gerar a assinatura hexadecimal da mensagem acima.
Você precisa incluir estes 3 headers em TODAS as requisições à API:

X-API-Key	Sua chave pública (public_key)	pk_live_xxxxxxxxxxxxx
X-API-Signature	Assinatura HMAC-SHA256 gerada	a1b2c3d4e5f6...
X-API-Timestamp	Timestamp em milissegundos	1705312800000

Crie uma nova transação PIX para receber pagamentos. Você receberá um QR Code e código PIX que o cliente pode usar para pagar. É simples: você cria a transação, recebe o QR Code, o cliente paga e você recebe a confirmação!

Valor Mínimo:
O valor mínimo para criar uma transação é R$ 7,11. Valores menores serão rejeitados com erro INVALID_AMOUNT.

Modo Sandbox
Ao enviar o header X-Sandbox: true, a transação será criada em modo de teste. Transações de teste não afetam saldos reais, não são processadas pelo gateway e são marcadas com is_test: true na resposta.
amount	number	Valor da transação (mínimo: R$ 7,11)	
description	string	Descrição da transação	
client_name	string	Nome do cliente	
client_document	string	CPF/CNPJ do cliente	
client_email	string	Email do cliente	
splits	array	Array de splits para divisão do pagamento	
Exemplo simples criando um pagamento de R$ 100,50:

{
  "amount": 100.50,
  "description": "Pagamento de produto XYZ"
}
Criando uma transação de teste (não afeta saldos reais, não chama gateway real):

{
  "amount": 100.50,
  "description": "Pagamento de teste",
  "sandbox": true
}
Ou usando o header: X-Sandbox: true

 Importante: Transações em modo sandbox não são processadas pelo gateway real, não afetam saldos e não geram créditos. Use apenas para testes de integração.

const crypto = require('crypto');

const crypto = require('crypto');

function ordenarObjeto(obj) {
  if (obj === null || typeof obj !== 'object' || Array.isArray(obj)) {
    return obj;
  }
  const ordenado = {};
  Object.keys(obj).sort().forEach(key => {
    ordenado[key] = typeof obj[key] === 'object' && obj[key] !== null && !Array.isArray(obj[key])
      ? ordenarObjeto(obj[key])
      : obj[key];
  });
  return ordenado;
}

function gerarAssinatura(method, path, body, timestamp, privateKey) {
  let bodyStr = '';
  if (body && Object.keys(body).length > 0) {
    const bodyOrdenado = ordenarObjeto(body);
    bodyStr = JSON.stringify(bodyOrdenado);
  }
  const message = `${method}${path}${bodyStr}${timestamp}`;
  return crypto.createHmac('sha256', privateKey).update(message).digest('hex');
}

async function criarPagamento(amount, description, publicKey, privateKey) {

Exemplo completo usando PHP com cURL:

<?php
function ordenarArray($array) {
    if (!is_array($array)) {
        return $array;
    }
    ksort($array);
    foreach ($array as $key => $value) {
        if (is_array($value) && !isset($value[0])) {
            $array[$key] = ordenarArray($value);
        }
    }
    return $array;
}

function gerarAssinatura($method, $path, $body, $timestamp, $privateKey) {
    $bodyOrdenado = ordenarArray($body);
    $bodyStr = json_encode($bodyOrdenado);
    $message = $method . $path . $bodyStr . $timestamp;
    return hash_hmac('sha256', $message, $privateKey);
}

function criarPagamento($amount, $description, $publicKey, $privateKey) {
    $timestamp = (string)(time() * 1000);
    $path = '/api/gateway/payments';
    $body = ['amount' => $amount, 'description' => $description];
    $signature = gerarAssinatura('POST', $path, $body, $timestamp, $privateKey);
    
    $ch = curl_init('https://nedpayapp.com/api/gateway/payments');
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        'Content-Type: application/json',
        'X-API-Key: ' . $publicKey,
        'X-API-Signature: ' . $signature,
        'X-API-Timestamp: ' . $timestamp
    ]);
    curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($body));
    
    $response = curl_exec($ch);
    $httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
    curl_close($ch);
    
    if ($httpCode !== 200) {
        throw new Exception('Erro na requisição: ' . $response);
    }
    
    return json_decode($response, true);
}

$PUBLIC_KEY = 'sua_public_key_aqui';

    echo 'Pagamento criado: ' . json_encode($resultado, JSON_PRETTY_PRINT);
} catch (Exception $e) {
    echo 'Erro: ' . $e->getMessage();
}

import requests
import hmac
import hashlib
import json
import time

def ordenar_objeto(obj):
    if isinstance(obj, dict):
        return {k: ordenar_objeto(v) for k, v in sorted(obj.items())}
    elif isinstance(obj, list):
        return [ordenar_objeto(item) for item in obj]
    return obj

def gerar_assinatura(method, path, body, timestamp, private_key):
    body_ordenado = ordenar_objeto(body) if body else {}
    body_str = json.dumps(body_ordenado, sort_keys=True, separators=(',', ':'))
    message = f"{method}{path}{body_str}{timestamp}"
    return hmac.new(
        private_key.encode('utf-8'),
        message.encode('utf-8'),
        hashlib.sha256
    ).hexdigest()

def criar_pagamento(amount, description, public_key, private_key):
    timestamp = str(int(time.time() * 1000))
    path = '/api/gateway/payments'
    body = {
        'amount': amount,
        'description': description
    }
    signature = gerar_assinatura('POST', path, body, timestamp, private_key)
    
    url = 'https://nedpayapp.com/api/gateway/payments'
    headers = {
        'Content-Type': 'application/json',
        'X-API-Key': public_key,
        'X-API-Signature': signature,
        'X-API-Timestamp': timestamp
    }
    
    response = requests.post(url, json=body, headers=headers)
    response.raise_for_status()
    return response.json()

PUBLIC_KEY = 'sua_public_key_aqui'
PRIVATE_KEY = 'sua_private_key_aqui'

try:
    resultado = criar_pagamento(100.50, 'Pagamento de teste', PUBLIC_KEY, PRIVATE_KEY)
    print('Pagamento criado:', json.dumps(resultado, indent=2, ensure_ascii=False))
except requests.exceptions.RequestException as e:
    print(f'Erro ao criar pagamento: {e}')
